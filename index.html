Elite Dangerous — Controller Binding Layouts

CMDR John DeParagon — WinWing HOTAS (Config A) & VirPil HOSAS (Config B)

Organized to match the in-game Controls menu — General → Ship → SRV, subcategories in menu order — so you can work straight down the binding screen. Two columns per binding: WinWing (active rig) and VirPil (FA-Off rig, ~Sept). Playing in VR (Quest 3). On-foot excluded (mouse + keyboard).

Legend: ⚠ = category placement or binding name I'm inferring (not yet confirmed against your current build) — verify on read-through or via Custom.binds. (unbound) = intentionally not bound. (spare) = free control available.

Standing hardware rules

  • Never bind Trigger stage 2 — a stage-2 pull also fires stage 1. Secondary Fire is on the stick's front-top Red Button instead.
  • Throttle is forward-only (no detent); reverse = Toggle Reverse Throttle. Set SimApp afterburner ratio so the AB detent doesn't eat range.
  • Firing Deploys Hardpoints = ON — hardpoints auto-deploy on fire, so the manual Deploy Hardpoints bind is a convenience.
  • Headlook stays fully unbound (anti-VR; headset handles look). VR re-center = Reset HMD Orientation, which lives in Ship → Miscellaneous, not Headlook.
